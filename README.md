# Map of the Zenith

A personal, time-based progress tracker built to enforce long-term consistency
towards GATE CSE & DA 2027.

This project visualizes daily and weekly progress across a year, emphasizing
honest execution over short bursts of intensity.

## What it does
- Displays a live countdown to GATE 2027
- Tracks year-level progress using real calendar dates
- Shows week-by-week milestones with locked/unlocked states
- Persists completed weeks locally using browser localStorage

## Why this exists
Long preparation cycles fail when progress becomes invisible.
This project exists to make progress *visible, accountable, and continuous*.

No logins. No backend. Just truth over time.

## Persistence
Completed weeks are stored locally in the browser using `localStorage`,
so progress is preserved across page refreshes.

- Persist daily logs
- Improve accessibility and mobile UX
- Add visual indicators for upcoming unlocks
